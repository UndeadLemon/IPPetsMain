IP Pets
This IP's Pet Project Info Suggestions Box Changelog

IP Pets is a little game/web toy inspired by the likes of NeoPets and Tamagotchi. Every IP Address has a pet, and that pet can get hungry. If that pet isn't fed...well, there will be a new pet at that IP Address soon enough.

More features planned, eventually, such as moods and other pet stats. Stay tuned!

The technologies used for this project include Python/Flask, SQL, the Zoo Animals API, Bootstrap, and of course, HTML and CSS!
